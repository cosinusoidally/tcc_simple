This is the web version of pnut_js. Drive it from your web browser devtools console.
In devtools run: 
go();
If this is successful then a link should appear to allow you to download the freshly built pnut.exe file. Download this file, rename it to pnut-exe, make it executable and place it in ../artifacts (the dir above the pnut_refactor dir). From the experiments dir you can then run: 
./mk_tcc-boot-mes
From the experiments dir in order to build tcc-boot-mes .
Tip, you can run a web browser on localhost for testing with: 
python3 -m http.server
IMPORTANT MAKE SURE YOU DISABLE CACHING IN THE NETWORK TAB OF YOUR BROWSER DEVTOOLS OR YOU MIGHT GET STALE VERSIONS OF CODE CAUSING CONFUSION DURING DEV
